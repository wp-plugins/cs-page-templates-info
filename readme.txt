== Description ==

A very simple plugin that provides information about the Custom Page Templates available in the theme: What they are, which are not used and where the used ones are used, selectively.

Main features:
Available page templates list
Unused page templates list
List of page templates with checkboxes that output when submitted the links where they are used

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the `cs_page_templates_info` directory with all it's contents to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How do You use it? =

You go to Settings -> Page Templates Info to see all it's settings.
There, you are able to see the available custom page templates of the theme, the unused ones and,
you are able to select which templates you want to see the used pages of by checking the box next to them and clicking "Click to see pages" button at the bottom.
A list of all the pages used by the selected page templates will appear.
